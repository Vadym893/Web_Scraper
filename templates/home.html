Ceneo Web Scraper

Project Description

The Ceneo Web Scraper is a Flask-based web application designed to scrape reviews and provide statistics for specialized products from the online shop Ceneo. It allows users to search for products, retrieve reviews from other users, view statistical analysis of the reviews, and access the history of recent searches.

Application Operation

The application utilizes web scraping techniques to extract data from Ceneo's product pages. It first requests the HTML content of the product page using the requests library. Then, it parses the HTML content using Beautiful Soup to extract relevant information such as product reviews, ratings, and descriptions. The extracted data is then processed and displayed to the user.

Libraries Used

  • Flask: Used for building the web application.
  • Pandas: Utilized for data manipulation and analysis.
  • JSON: Used for handling data in JSON format.
  • io: Used for handling file input and output operations.
  • Math (a bit): Used for basic mathematical operations.
  • Beautiful Soup: Used for parsing HTML content.
  • re: Used for regular expression matching.
  • Requests: Used for making HTTP requests to fetch web pages.

Applied Approaches

The application follows these steps to perform its operations:

  1. User inputs the name or identifier of the product they want to search for.
  2. The application sends a request to the Ceneo website, searching for the specified product.
  3. It scrapes the product page for reviews, ratings, and other relevant information using Beautiful Soup.
  4. The scraped data is processed and analyzed using Pandas and math libraries to generate statistics.
  5. The results are displayed to the user, including a summary of reviews, average rating, and recent search history.
